## 请假类型

| 类型 | 额度 | 是否带薪 | 提前申请 |
|---|---|---|---|
| 年假 | 每年 5-15 天（按司龄） | 是 | 提前 3 个工作日 |
| 病假 | 按医院证明 | 部分带薪 | 当天可补 |
| 事假 | 无固定额度 | 否 | 提前 1 个工作日 |
| 调休 | 按加班时长 | 是 | 提前 1 个工作日 |

## 请假流程

1. 在飞书「审批」中选择对应请假类型。
2. 填写起止时间、事由，病假需上传医院证明。
3. 提交后由直接主管审批；超过 3 天需部门负责人二级审批。
4. 审批通过后，考勤系统自动扣减对应额度。

## 常见问题

**Q：年假没休完会清零吗？**
A：当年未休年假可顺延至次年 3 月 31 日，逾期清零。

**Q：请假当天临时生病来不及申请怎么办？**
A：可先口头/消息告知主管，事后 1 个工作日内补交病假审批与证明。

**Q：调休有效期多久？**
A：加班调休需在 3 个月内使用，逾期作废。
